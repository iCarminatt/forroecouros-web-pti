# Forró & Couros — Website

Bem-vindo ao repositório do site demonstrativo do ateliê "Forró & Couros" — uma página estática construída para a disciplina Web Standards e para demonstrar um layout responsivo com imagens reais, tipografia refinada e interatividade mínima em JavaScript.

Demo ao vivo: https://pti-webdev-git.netlify.app/

Resumo rápido
- Layout responsivo com Grid e Flexbox
- Telas: Hero, Sobre, Serviços, Galeria, Contato e Rodapé
- Assets otimizados: imagens em WebP/JPG no diretório `images/`
- Interatividade leve: menu mobile, animações via IntersectionObserver e barra de progresso

Como usar
1. Clone o repositório:

	git clone https://github.com/iCarminatt/forroecouros-web-pti.git

2. Entre na pasta e sirva localmente:

	cd forroecouros-web-pti
	python -m http.server 8000

3. Abra `http://localhost:8000` no navegador.
